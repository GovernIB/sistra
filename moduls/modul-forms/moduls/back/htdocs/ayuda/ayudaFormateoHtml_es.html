En las etiquetas se permite introducir los siguientes tags HTML para formatear el texto:

  • HTML entities (p.e. &euro;)
  • Negrita: <strong></strong> o <b></b>
  • Subrayado: <u> </u>
  • Cursiva: <em> </em> o <i> </i>
  • Tamaño pequeño: <small></small>
  • Tamaño grande: <big></big>
  • Superindice: <sup></sup>
  • Enlaces: <a href="url" target="_blank"> </a>

En el componente Label para las etiquetas de tipo mensaje se pueden usar los siguientes tags para formatear el mensaje:

  • Parrafos: <p> </p>
  • Saltos línea: <br/>
  • Listas: <ul> </ul> y <li> </li>

Se debe respetar estrictamente el formato de los tags (respetando espacios en blanco, minúsculas, comillas, etc.). Si no se sigue estrictamente el formato aparecerá el texto sin aplicar el tag.
