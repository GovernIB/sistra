Desde esta pantalla podrá dar de alta cuadernos de carga. Además se podrán adjuntar nuevos ficheros al cuaderno, así consultar y descargar cada uno de los ficheros que componen el cuaderno.

Se podrá modificar el cuaderno y dar de alta nuevos ficheros cuando el cuaderno se encuentre en estado Pendiente de envio.

Cuando se adjuntes ficheros de importación de trámites, dominios o formularios, el nombre de dichos ficheros deberán cumplir el siguiente formato:

  • dominio-[ID_DOMINIO].xml Para ficheros de importación de dominios
  • tramite-[ID_TRAMITE]-[VERSION_TRAMITE].xml Para ficheros de importación de trámites
  • form-[MODELO_FORM].xml Para ficheros de importación de formularios

    • Cuando un cuaderno ha sido dado de alta, se puede enviar a auditar desde el listado, tras lo cual pasará al estado Pendiente auditar si requiere auditoría o No requiere auditoría si los ficheros que conforman el cuaderno de carga no contienen ningún script o dominio que deban ser auditados.

    Cuando un cuaderno haya sido auditado, y por tanto se encuentre en estado Auditoría aceptada, se podrá marcar como importado con lo cual el cuaderno pasará a la lista de cuadernos finalizados.
