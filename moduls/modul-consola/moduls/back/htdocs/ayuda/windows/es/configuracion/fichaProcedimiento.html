CONFIGURACIÓN DEL PROCEDIMIENTO

Esta ventana contiene las opciones de configuración del nuevo procedimiento o del procedimiento seleccionado para edición.

DESCRIPCIÓN DE CAMPOS

Los campos que aparecen en la ventana son los siguientes:
  • Identificador: Código que identifica unívocamente al procedimiento. Este código debe ser un alfanumérico de 20 posiciones y es recomendable que sea un código nemotécnico
  • Descripción: Texto descriptivo del procedimiento. Este texto debe estar formado por un máximo de 100 carácteres alfanuméricos que describan el procedimiento en cuestión
  • Organismo: Este campo muestra el organismo en el que se enmarcará el procedimiento. Si pulsa el botón ... se le mostrará una nueva ventanta con el listado de organismos existentes, y podrá asociar un organismo al procedimiento en cuestión.
  • Gestión expedientes: Este apartado contiene las configuraciones del procedimiento relativas a la gestión de expedientes. El detalle de cada campo es el siguiente:
    • Unidad Administrativa: Este campo permite establecer la Unidad Administrativa por defecto que se utilizará en la creación de notificaciones asociadas al procedimiento. Este valor será modificable por el gestor al dar de alta una nueva notificación
    • Aviso notificaciones a gestores: Este check habilita la posibilidad de que los gestores de un procedimiento reciban avisos sobre las notificaciones asociadas al mismo
    • Permitir SMS: Este check habilita la posibilidad de enviar avisos SMS al ciudadano al dar de alta una notificación o una comunicación asociada al procedimiento
  • Gestores: Este campo contiene un listado de los gestores que tienen permisos sobre el procedimiento
