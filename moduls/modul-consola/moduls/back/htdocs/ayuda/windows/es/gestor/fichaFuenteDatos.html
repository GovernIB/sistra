ALTA / EDICIÓN DE FUENTES DE DATOS DEL PROCEDIMENTO

Esta pestaña de la ventana de procedimiento permite llevar a cabo un mantenimento de las fuentes de datos asociadas a un procedimento.

OPERATIVA

Las acciones relativas a las fuentes de datos que se pueden realizar en esta pestaña son:
  • Crear nueva fuente de datos: Pulsando el botón Añadir puede dar de alta una nueva fuente de datos asociada al procedimento.
  • Borrar una fuente de datos existente: En caso que el procedimento tenga fuentes de datos asociadas, podrá seleccionar una y eliminarla pulsando el botón Borrar
  • Editar fuente de datos existente : En caso que el procedimento tenga fuentes de datos asociadas, podrá seleccionar una y editarla pulsando el botón Editar
