ALTA / EDICIÓN DE GESTOR DE FORMULARIO

Esta ventana contiene las opciones de configuración del nuevo gestor de formulario o del gestor de formulario seleccionado para edición. Las opciones de configuración se refieren a identificador y las urls envueltas en la comunicación entre SISTRA y el gestor de formularios.

DESCRIPCIÓN DE CAMPOS

Los campos que aparecen en la ventana son los siguientes:
  • Identificador: Código del gestor de formularios. Este código permite identificar al gestor de formulario en los diferentes puntos de SISTRA
  • Descripción: Texto alfanumérico con un máximo de 100 caracteres que describe brevemente el gestor de formularios
  • URL Gestor: Este campo ha de contener la Url base del gestor de formularios. En caso de que el gestor de formularios esté instalado en el mismo servidor que SISTRA se puede referenciar a la variable @sistra.url@
  • URL Tramitación: En este campo se ha de consignar la url que será invocada por SISTRA para pasar los datos al gestor de formularios. Esta url ha de ser absoluta. Al igual que en el campo anterior, se pueden referenciar variables como la url base (@forms.server@)
  • URL RedirecciÓn: En este campo se ha de cumplimentar la url a la que se redigirá el navegador para redirigirse al gestor de formularios. En caso de que el gestor de formularios esté instalado en el mismo servidor que SISTRA, esta url puede ser relativa. Esta url también puede contener variables como la url base (@forms.server@)
