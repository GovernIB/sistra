MODELS DE DOCUMENT D'ORGANISME

En aquesta pipella de la pantalla d'organisme es presenta un llistat amb els models de documents associats a l'organisme.
Al llistat es visualitcen el codi i descripció dels models de documents.
El llistat està paginat, i es pot ordenar per totes les columnes.

OPERATIVA

Les accions que es poden realitzar poden variar en funció del rol i l'entorn. Les accions disponibles són les següents:
  • Cercar: Cerca models de documents podent filtrar mitjançant un camp de filtrat.
    El camp de filtrat cerca coincidències a l'identificador i a la descripció del model de documents. No és sensible a les majúscules i minúscules.
    Perquè es refresqui el llistat amb les condicions de filtrat s'ha de pitjar el botó Cercar.
    Opció disponible per qualsevol rol o entorn.
  • Nou model de document: Pitjant el botó Nou s'obri la pantalla per donar d'alta un nou model de document.
  • Edició de model de document: Seleccionant un model de document del llistat i pitjant el botó Editar, s'obri la pantalla de model de document amb la informació del model.
  • Consulta de models de document: Seleccionant un model del llistat i pitjant el botó Consultar, s'obri la pantalla de model de document amb la informació del domini en mode consulta.
  • Eliminació de model de document: Seleccionant un model del llistat i pitjant el botó Eliminar, s'eliminarà el model seleccionat.
Pitjant el botó dret apareix un menú contextual amb les accions disponibles.
Les opcions Edició i Consulta no estan disponibles a la vegada.
Fent doble click damunt qualsevol registre, es realitza l'acció Edició o Consulta, segons el rol i l'entorn.

DESCRIPCIÓ DE CAMPS

Els camps que apareixen al llistat són els següents:
  • Codi: Identificador del model de documents.
  • Descripció: Descripció del model de documents.
