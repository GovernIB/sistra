Sintaxis
Para referenciar a un campo de un formulario dentro de un campo de tipo referencia hay que utilizar la siguiente sintaxis:

identificadorFormulario.instancia.nombreCampo

Ej: FORM1.1.DATOS_PERSONALES.NOMBRE

Para las listas de valores podemos acceder para la opción seleccionada tanto al código como al texto.

Ej. : Supongamos un campo que tiene un desplegable con localidades

FORM1.1.DATOS_PERSONALES.LOCALIDAD : accedemos al texto de la localidad
FORM1.1.DATOS_PERSONALES.LOCALIDAD[CODIGO] : accedemos al código de la localidad
