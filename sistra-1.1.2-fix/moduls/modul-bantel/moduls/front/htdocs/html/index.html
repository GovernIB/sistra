Ajuda general

Con esta opción podremos recuperar la Clave de Tramitación para aquellos trámites con autenticación Anónima.

Se buscará a partir del NIF y del Código del Trámite. Buscará en las Tablas de Persistencia, Backup de Persistencia, Entradas Telemáticas y Pre-Registro.

En el caso de Persistencia y Backup de Persistencia buscará el NIF entre los datos del Formulario.

Aquesta opció ens permet crear un nou expedient:

  • a partir d’un número d’entrada a la bandeja Telemàtica
  • creant-lo de nou
