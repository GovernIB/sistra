Campo
Descripción
Tipo Indica si el documento es un Formulario, Anexo o Pago
Identificador Identificador del documento. Debe ser único entre los documentos del trámite.
Orden de presentación Orden de presentación en el asistente
Modelo Indica el modelo del documento. Todo documento debe estar definido en el RDS mediante un modelo / versión. La versión se particulariza según nivel de autenticación.
Identificador PAD Si el documento puede estar en la PAD se indicará el identificador interno de la PAD para ese documento. En la tramitación se comprobará si el usuario tiene registrado dicho documento en la PAD y le dará opción a reutilizarlo.
Presentar en Preregistro Para formularios en caso de que el trámite requiera que se realice de forma presencial la plataforma indicará al usuario que debe imprimir el formulario y llevarlo firmado.
Justificante en Preregistro Para formularios se puede indicar que uno de los formularios se utilice como justificante en preregistro.
Presentar

Para anexos se puede indicar que el usuario tenga que anexarlo telemáticamente o bien entregarlo de forma presencial.
Extensiones posibles Para anexos indica la lista de extensiones posibles del documento. Esta lista estará compuesta por sufijos de tres o cuatro letras separados por "," (Ej; txt,doc,zip)
Conversión a PDF

Si se habilita esta opción el fichero anexado se convertirá a PDF. Tras el proceso de anexado se mostrará un mensaje indicando que se ha realizado esta conversión y que se debe verificar el contenido del documento.

No se convertirán todas la extensiones, sólo las siguientes: doc, docx y odt.

Esta opción es incompatible con indicar que se debe firmar el documento. Si se indica que se debe firmar el documento no se realizará la conversión.

Tamaño máximo Para anexos indica el tamaño máximo que puede tener el documento (en Kb)
Url Plantilla Para anexos se puede indicar que el documento deba rellenarse según una plantilla. En este caso se indicaría la URL de la plantilla. La plataforma mostrará un enlace a esta URL. Se recomienda que la plantilla sea un fichero zip para evitar que el navegador abra directamente ficheros tales como documentos de word.
Instrucciones descargar plantilla

En caso de haber rellenado el campo anterior se puede indicar al usuario en el paso inicial "Debe saber" que debe descargarse la plantilla antes de iniciar el trámite.
Compulsar

Para anexos se puede indicar que el usuario debe mostrar presencialmente el original del documento para su compulsa. Si existe un documento con estas características convertirá la tramitación en presencial.
Presentar Fotocopia

Para anexos se puede indicar que el usuario debe mostrar presencialmente una fotocopia. Si existe un documento con estas características convertirá la tramitación en presencial.
Genérico

Para anexos permite que el usuario anexe más de una instancia de un documento. Para cada instancia se permitirá que el usuario introduzca una descripción personalizada para la instancia.
Si es genérico no se puede indicar que sea fotocopia.
Max. numero doc. genericos En el caso de genérico indica el máximo número de instancias.
Nombre documento Nombre descriptivo del documento
Información adicional Información específica del documento.
