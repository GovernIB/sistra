Camp
Descripció
Tipus Indica si és un camp o separador de blocs.
- Camp : Permet referenciar al valor d’un camp d’un formulari. Es deurà indicar una descripció i com valor una referència al camp d’un formulari o bé indicar un script per al càlcul del valor.
- Separador: Serveix per a separar en seccions diferents camps. Únicament s’indicarà una descripció.
Camp del formulari Per a tipus camp s’indicarà la referència d’un camp d’un formulari.
Script valor Per a tipus camp s’indicarà un script que realitzarà el càlculo del valor.
Script visibilitat

Permetrà indicar si es deurà generar aquesta dada. El tipus d’aquest camp és script i espera com resultat: S o N.

Per a aquest script està disponible el plugin de Pagaments (PLUGIN_PAGOS) que permet consultar informació dels pagaments.

Ej: PLUGIN_PAGOS.getDatoPago("PAGO1", 1,"DATOS_PAGO.TIPO");
